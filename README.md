# 水浒人物性格测试

一个基于网页的水浒传人物性格测试，测测你在梁山泊更像哪位好汉。

## 在线体验

https://imerance123.github.io/shuihu-test/

## 功能特点

- **40 道精心设计的题目**，涵盖处事抉择、危机应对、人际关系、价值取向、自我认知、江湖态度
- **6 维性格模型**：忠义、胆勇、智谋、正道、自由、聚义
- **24 位水浒好汉**匹配，每人配有深度解读
- **Canvas 雷达图**直观展示六维倾向
- **相近/互补/对立**三种关系分析，帮助你理解自己与他人的互动方式
- 经典语录、优势盲点、团队角色、成长建议、人生启示
- 结果一键复制分享

## 技术实现

纯前端静态网站，无需后端服务：

- `index.html` — 页面结构
- `data.js` — 维度定义、24 位人物数据、40 道题目
- `results.js` — 24 位人物的详细结果文案
- `scoring.js` — 基于欧氏距离的匹配算法
- `app.js` — 应用逻辑与雷达图绘制
- `styles.css` — 水墨风主题样式

## 匹配算法

1. 每道题的选项对各维度施加权重（正/负/零）
2. 累计原始分数后归一化到 0-100
3. 计算用户向量与 24 位人物向量的欧氏距离
4. 距离越小相似度越高，取相似度前 3 名作为匹配结果

## 本地运行

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。
